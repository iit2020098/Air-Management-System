Welcome to the IIITA !!

User Manual of Air Management System

Pre-requisites for project: -

1) Intellij IDEA need to be installed in the system
2) Java need to be installed in the system


For executing the project , Follow the given steps : 

Step-01: Un-zip the folder.

Step-02: Open the mini_project.

Step-03: Open the final_project in the Intellij IDEA.

Step-04: Click the “src” folder and open it.
 
Step-05: Now Click “airmanagement” folder.
 
Step-06: Now click the “HomeScreen” and then click “HomeScreen.java” file.

Step-07: Run the “HomeScreen.java” file. This will start the program.
